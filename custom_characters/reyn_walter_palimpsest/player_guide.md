# Рейн Вальтер «Палимпсест» - Гайд игрока (после правок ГМа)

---

## 1) Базовая формула хода

`Угроза -> Короткая команда -> Способность -> Проверка ресурса -> Передача хода`

## 2) Три паттерна речи (без бонусов)

### Архитектор
- Когда: анализ, план, распределение ролей.
- Реплики: "Фиксирую вводные." / "Работаем по шагам."

### Зеркало
- Когда: деэскалация и контакт с союзниками.
- Реплики: "Сначала выслушаем." / "Нам нужен рабочий вариант."

### Предел
- Когда: критическая боевая фаза.
- Реплики: "Спор после боя." / "Цель в приоритете."

Важно: это только стиль отыгрыша, не механические стойки.

## 3) Боевой цикл

1. Назови цель хода (удержать, поднять, дожать).
2. Объяви способность OOC.
3. Если используешь укол — проверь ампулы.
4. Дай короткую IC-фразу.
5. Передай ход конкретному союзнику.

## 4) Расходники для укола

Перед сессией проверь:
1. Есть ли инъекционный набор.
2. Есть ли 2 ампулы (рекомендованный минимум).

Шаблон:
`[OOC] Трачу 1 ампулу, использую Стабилизационный укол на <союзник>.`
`[IC] "Держись, я закрыл критичный сбой."`

## 5) Работа с душами (кратко)

Используй как инструмент информации и оценки риска, а не как постоянный боевой буст.

Шаблон:
`[OOC] Использую спиритуализм для контакта/ауры.`
`[IC] "Мне нужен чистый след."`

## 6) Чего избегать

1. Не объявляй режимы как источник механических бонусов.
2. Не забывай расходники на укол.
3. Не отбирай инициативу у других игроков длинными директивами.
4. Не заявляй эффекты, которых нет в листе/согласовании с ГМом.

## 7) Чеклист перед сессией

1. Паттерн речи на старт.
2. 2-3 короткие командные реплики.
3. Наличие инъекционного набора и ампул.
4. Короткая личная цель на сессию.
