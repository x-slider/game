## О сеттинге

**"Башня Возвышения"** - настольная ролевая игра по системе D&D 5e с уникальным сеттингом.

### Влияния
- **Warhammer 40k** - Бог-Император, иерархия, эстетика Примархов
- **Mass Effect** - Спектры, Совет Цитадели, N7
- **Классическое D&D** - классы, расы, система

### Особенности мира
- **Империя** с детальной иерархией (Сыны Империи, Спектры, Рыцари)
- **Бог-Император** (рост 6м) как центральная фигура
- **Совет Цитадели** - политический орган (7 членов)
- Смесь магии и технологий (мехи, химеры, огнестрел)
- Морально серый сеттинг
- Развитая система душ и спиритуализма

### Фракции

| Фракция | Описание |
|---------|----------|
| Империя | Основное государство |
| Орден вольных стрелков | Наёмники-снайперы |
| Пылающий авангард | Личная гвардия Гезария |
| Синие Светила | Стража Иллиума |
| Шепчущий ветер | Клан табакси |
| Ваалиты | Вампироподобные существа |

### Действующие персонажи игроков

1. **Восьмая** - Магус/Демиург, химера (Catherine Rauch)
2. **Хельм Деклар** - Жрец, аасимар (Vadim Osipov)
3. **Хори** - Волшебник-хронург, калаштар (Elena Rabbit)
4. **Блик** - Плут, табакси (Kirill Yakimenko)
5. **Касуми Аракава** - Чародей, кицунэ (Varvara Levina)
6. **Эдем Калхер** - Инженер, полуэльф (Vadim Osipov)
